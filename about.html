About FastAidNet

FastAidNet is a First Responder Volunteer Network designed to provide immediate emergency assistance. The system allows users to send emergency requests, receive quick responses from volunteers, and track help in real-time.

How It Works:

  1. Users report emergencies via the system.
  2. Nearby volunteers receive notifications.
  3. Responders take action and update the status.
  4. Users can track response history.
